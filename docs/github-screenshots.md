# Documentación de Capturas GitHub

## Instrucciones para Capturar Imágenes de Commits

### 1. Commits Branch `develop`
- Navegar a: https://github.com/JuanOlguinDuoc/levelup-kotlin/commits/develop
- Capturar pantalla mostrando el historial de commits
- Guardar como: `commits-develop.png`

### 2. Commits Branch `master`  
- Navegar a: https://github.com/JuanOlguinDuoc/levelup-kotlin/commits/master
- Capturar pantalla mostrando el historial de commits
- Guardar como: `commits-master.png`

### 3. Commits Branch `feature/test2`
- Navegar a: https://github.com/JuanOlguinDuoc/levelup-kotlin/commits/feature/test2
- Capturar pantalla mostrando el historial de commits
- Guardar como: `commits-feature.png`

### 4. Git Workflow Diagram
- Crear un diagrama del Git Flow usado en el proyecto
- Mostrar las ramas: master, develop, feature/*, hotfix/*
- Guardar como: `git-workflow.png`
